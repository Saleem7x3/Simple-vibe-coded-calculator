# SimpleScientificCalculator

A production-grade Android mobile application prototype for a simple scientific calculator. It adheres to the MVVM architectural pattern, utilizing Jetpack Compose for the UI, Kotlin for the backend logic, and Hilt for dependency injection (mocked for this prototype). The app features a robust expression evaluator capable of handling basic arithmetic operations, parentheses, and common scientific functions like sine, cosine, tangent, square root, logarithm (base 10), and natural logarithm.

## Tech Stack
kotlin

## Build Instructions
1. Ensure you have Android Studio installed (Jellyfish | 2023.3.1 or newer recommended).
2. Unzip the provided project files into a directory.
3. Open Android Studio and select 'Open an existing Android Studio project'.
4. Navigate to the unzipped project directory and select it.
5. Android Studio will automatically sync the Gradle files. If prompted, update Kotlin plugin or Gradle to the recommended versions.
6. Once Gradle sync is complete, select an Android emulator or connect a physical Android device.
7. Click the 'Run' button (green triangle icon) in the toolbar to build and deploy the application to your selected device/emulator.
8. The app should launch on the device/emulator, displaying the scientific calculator interface.
